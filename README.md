# Whaler

This app depends on [docker](https://www.docker.com/).

![Whaler](whaler.png)

## Setup

### Linux & Darwin/OSX

```sh
$ curl -sSL https://git.io/get-whaler | sh
```

OR

```sh
$ curl -sSL https://git.io/get-whaler | sh -s -- [options]
```

Available options:

```
--version=<VERSION>    Specify an "Whaler" version (dev, latest, ...).
```
