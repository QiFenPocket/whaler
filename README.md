# Whaler

## Install

```sh
$ curl -sSL https://raw.githubusercontent.com/cravler/whaler/master/.docker/install.sh | sudo sh
```

OR

```sh
$ curl -sSL https://raw.githubusercontent.com/cravler/whaler/master/.docker/install.sh | sudo sh -s -- --version=latest
```

## Run app

```sh
$ whaler
```
