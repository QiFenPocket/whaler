# Whaler

![Whaler](whaler.png)

## Install

```sh
$ curl -sSL https://git.io/whaler-install | sudo sh
```

OR

```sh
$ curl -sSL https://git.io/whaler-install | sudo sh -s -- --version=latest
```

## Run app

```sh
$ whaler
```
